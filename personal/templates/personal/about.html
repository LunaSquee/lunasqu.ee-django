Me as a coder

I have been writing code since the age of 10. I got into coding thanks to a game called ROBLOX, where you build things with blocks and use the lua scripting language to do neat stuff with your builds.

I got into web design at the age of 12 thanks for a ROBLOX fan site called LuaLearners, which made me want to create my own site. So, first I learned HTML on my own by looking at the site's code (using view-source) and it seemed pretty easy to me. My first website was hosted on a free web hosting service (I can't remember the name of it) and it didn't have much to it. After that I got into the basics of PHP to add some functionality to my site (A login system, which was rather terrible and used a lot of resources) and I didn't know much about JavaScript back then.

I can currently code well in HTML5, JavaScript, Python and Java. I also know the basics of C++.
